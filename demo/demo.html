fnlCarousel Demo

The DOM hierarchy for the elements to be used by fnlCarousel is easy: outer div, relative positioned, first child of that div is of class .container, items can be named and styled as you want, just remember to add the class name in the callback function that you provide to fnlSlider for item rendering.
Classes button, left, right, pageButton and curPage are used to style navigation buttons.
